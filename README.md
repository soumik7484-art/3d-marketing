## ūüé† Overview

**TOONHUB** is a full-viewport, animated 3D character figurine carousel built as a premium marketing hero section. Four vibrant figurines rotate through **center / left / right / back** roles, each with their own depth-based blur, scale, and opacity ‚ÄĒ all crossfading simultaneously over a matching background color.

> Designed to be a **drop-in hero section** for any 3D toy, collectible, or character brand.

<br/>

## ‚ú® Features

| Feature | Details |
|---|---|
| ūüé® **Dynamic Background** | Each figurine has its own color palette; background crossfades smoothly on navigate |
| ūüĆÄ **Depth Carousel** | 4-slot ring: center (hero) ¬∑ left ¬∑ right ¬∑ back ‚ÄĒ each with unique scale, blur & opacity |
| ‚ö° **650ms Crossfade** | All properties (position, scale, blur, opacity, background) animate with `cubic-bezier(0.4,0,0.2,1)` |
| ūüďĪ **Fully Responsive** | Mobile-first breakpoints at 640px ‚ÄĒ sizes, positions and text all adapt |
| ‚Ć®ÔłŹ **Keyboard Navigation** | `‚Üź` / `‚Üí` arrow keys navigate the carousel |
| ūüĖľÔłŹ **Image Preloading** | All 4 images preloaded on mount via `new Image()` for zero-flash transitions |
| ūüĆĺ **Grain Overlay** | SVG `fractalNoise` texture adds a tactile, premium print feel |
| ūüĒ† **Dual Typography** | **Anton** (display ghost text + discover link) ¬∑ **Inter** (UI labels + body text) |
| ūüĒí **Animation Lock** | `isAnimating` guard prevents double-firing during 650ms transition window |
